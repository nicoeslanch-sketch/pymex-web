Guía de edición

Cómo cambiar imágenes, textos y formularios.

Esta página está pensada para ayudarte a mantener el sitio sin perderte.

1. Cambiar imágenes de las planillas

Guarda tus capturas dentro de assets/productos/nombre-del-producto/ y luego reemplaza las rutas en assets/products.js.

2. Cambiar textos del sitio

Edita los archivos HTML para texto de página y assets/script.js para tarjetas o navegación común.

3. Conectar lista de espera a base de datos

El formulario hoy guarda en localStorage. La siguiente integración recomendada es Supabase, Airtable o un endpoint propio.

4. Publicar cambios

Guarda, luego ejecuta git add ., git commit -m "actualización" y git push.

5. Qué botones llevan a la plataforma futura

Los CTA de “Plataforma IA”, “Lista de espera” y “robot / análisis” apuntan a plataforma.html.
