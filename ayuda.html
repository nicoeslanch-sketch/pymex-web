Cómo usar esta web
y cómo cargar imágenes

Cómo agregar tu logo

Ya dejé el logo integrado usando el archivo que subiste. Si después quieres cambiarlo, solo reemplaza el archivo assets/logo-analixia-pyme.png por uno nuevo con el mismo nombre.

Cómo agregar imágenes reales a cada Excel

Guarda tus capturas dentro de la carpeta correspondiente, por ejemplo:

assets/productos/flujo-caja/1.png
assets/productos/flujo-caja/2.png

Luego edita el archivo assets/products.js y reemplaza el array images del producto correspondiente.

Cómo funciona la lista de espera

La parte visual ya quedó lista. Para que el correo se guarde realmente en una base de datos y se envíe una respuesta automática, necesitas conectar el formulario a un servicio.

Opción simple: Formspree
Opción más seria: Supabase + Resend
Opción intermedia: Google Apps Script

En assets/products.js ya te dejé un campo llamado WAITLIST_ENDPOINT. Ahí puedes poner la URL del endpoint cuando la tengas.
