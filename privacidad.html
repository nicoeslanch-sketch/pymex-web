Privacidad

Tratamiento de
datos básicos

Analixia Pyme puede recopilar datos básicos entregados voluntariamente por el usuario, como correo electrónico, para fines de contacto, lista de espera, seguimiento comercial o envío de novedades.

1. Datos que se recopilan

Por ahora el sitio contempla principalmente el ingreso voluntario de correo electrónico en formularios de interés.

2. Finalidad

Los datos se utilizan para responder solicitudes, registrar interesados, enviar avisos relacionados con productos y comunicar novedades de la marca.

3. Conservación y acceso

Los datos se mantendrán en sistemas razonables de almacenamiento y no deberían compartirse con terceros ajenos a la operación del servicio, salvo requerimiento legal o herramientas necesarias para operar la web.

4. Solicitudes del usuario

El usuario puede solicitar dejar de recibir correos o pedir la eliminación de sus datos de listas comerciales cuando corresponda.

5. Cambios

Esta política puede ajustarse a medida que el sitio incorpore nuevas integraciones como pagos, base de datos o automatizaciones.

También es un texto base general y editable. Si luego conectas el sitio con servicios externos, conviene actualizar esta política.
