ANALIXIA PYME — SITIO ESTÁTICO

Archivos principales:
- index.html
- tienda.html
- producto.html
- nosotros.html
- legal.html
- terminos.html
- privacidad.html
- reembolsos.html
- ayuda.html

Cómo usar:
1. Abre esta carpeta en VS Code.
2. Si quieres cambiar el logo, reemplaza:
   assets/logo-analixia-pyme.png
3. Si quieres cambiar o sumar capturas de tus Excels:
   - guarda las imágenes en assets/productos/<carpeta-del-producto>/
   - edita assets/products.js
4. Si quieres que la lista de espera guarde correos de verdad:
   - crea un endpoint
   - pega su URL en ANALIXIA_CONFIG.WAITLIST_ENDPOINT dentro de assets/products.js

Sugerencia para publicar:
- sube toda la carpeta a GitHub
- publica con Vercel o GitHub Pages
